## 📌 Overview

This is a 4th-year undergraduate final project that applies a **Long Short-Term Memory (LSTM)** deep learning model to forecast Kenya's agricultural production trends using over **60 years of historical data (1961–2021)**. The dataset is sourced from FAOSTAT and covers **150 crop types** across **18,182 records**, with analytical focus on five major crops: **Maize, Tea, Coffee, Wheat, and Oranges**.

**Core Research Question:**
> *Can a deep learning LSTM model accurately predict Kenya's agricultural production seasonal trends, and how well does it capture nonlinear, climate-driven patterns that traditional methods miss?*
